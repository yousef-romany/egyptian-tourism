## üìù Quick Entry Tips

### For JSON Fields in Strapi:
1. Click on the JSON field
2. Copy the entire JSON block (including `[` and `]`)
3. Paste directly into Strapi
4. Strapi will format it automatically

### For Images:
- Use high-quality images (at least 1200x800 px)
- Accepted formats: JPG, PNG
- Recommended: Search for "Egypt tourism photos" or use placeholder images initially
- Upload via the media library in Strapi

### Publishing Workflow:
1. Fill in all required fields
2. Click **"Save"** (top right)
3. Click **"Publish"** (top right)
4. Verify status shows "Published" (not "Draft")
